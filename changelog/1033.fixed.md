Fixed two GPU-preprocessing divergences from the CPU reference: `TorchSoftClipOutliers` silently skipped outlier clipping when predicting a single sample in KV-cache mode (predictions depended on test batch size), and `TorchAddSVDFeaturesStep` added an SVD column for single-feature datasets where the CPU pipeline adds none (predictions differed between `ENABLE_GPU_PREPROCESSING` on and off).
